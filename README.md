# web-dev-methods-project
Web Development Methods Project required for Graduate students in the CS 7830 Web Application Development II course
### React Frontend start up:
-	`cd frontend/insurance_client`
-   `npm run dev`
-   http://localhost:5173/
### Ruby on Rails Backend start up:
-	`cd backend/insurance_api`
-	`rails server`
-   http://localhost:3000/
### MySQL (Homebrew) start up:
-	`brew services start mysql`
-	`brew services list`
-	`brew services stop`
